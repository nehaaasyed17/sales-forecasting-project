# AI Sales Forecasting App

This project is an AI-based sales forecasting web application built using Python and Streamlit.

## Features
- Load and analyze Walmart sales data
- Visualize sales trends using graphs
- Prepare data for forecasting
- Simple and interactive web interface

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

## How to Run the Project
1. Install required libraries:
pip install streamlit pandas matplotlib scikit-learn


## Dataset
Walmart Sales Dataset (Kaggle)
